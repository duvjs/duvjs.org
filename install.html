安装

安装

环境要求

  1. node >= 8.0
  2. 小程序开发工具
    • 微信小程序开发工具：下载地址
    • 百度智能小程序开发工具：下载地址

安装vue-cli

npm install -g vue-cli

创建一个附带简易demo的初始化项目

vue init duvx/duv-init helloworld

按照提示安装依赖并运行

    cd helloworld
    npm install
    npm run bd

运行小程序

使用小程序开发工具打开项目目录下dist文件夹中的文件

  • dist/bd文件夹：百度智能小程序编译后的文件
  • dist/wx文件夹：微信小程序编译后的文件
